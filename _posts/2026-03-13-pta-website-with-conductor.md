Following on from my [previous post]({{ "/2026/03/12/conduct-your-ai-agents/" | relative_url }}), I have to say — the joy of actually installing and using [Conductor.build](https://conductor.build) has only grown.

One of the committee members of my daughter's school mentioned that the [Parents Teachers Association website](https://aldryngtonpta.com) needed an upgrade to modern standards. I thought — now that I'm in love with Conductor and can work on multiple projects and repositories at the same time — how can I best leverage it?

So I decided to build a [brand new website](https://nanda-nainadurai.github.io/aldryngton-pta-new/) for the PTA.

It was more or less one-shotted. Well, maybe three or four shots — but the result came out really well. There were quite a few pages of content, sub-pages, links, and PDF documents covering things like the school's treasury reports. Everything has now been ported into a GitHub repository, and Conductor even helped build a CMS to manage the content.

There was some additional work required from me to authenticate the CMS using GitHub authentication and restrict access to only specific usernames for security reasons — but all of that has now been done, which is great.

I've shared the new site with one of the committee members and we're in conversation about whether it can replace the existing one. Let's see how it goes.

If you have not tried [Conductor.build](https://conductor.build) yet, I strongly urge you to give it a go. Two years ago I tried Cursor and it was just amazing — now everyone talks about Cursor. I believe the same is going to happen for Conductor pretty soon.
